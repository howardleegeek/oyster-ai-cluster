# ClawVision PRD - 众包视觉数据平台

> **Version:** 1.0  
> **Date:** 2026-02-14  
> **Project:** ClawVision  
> **Goal：** 利用 ClawGlasses 让用户众包上传图片，构建大视觉模型训练数据集

---

## 1. 产品概述

**产品名称：** ClawVision  
**产品定位：** 众包视觉数据平台 + 商业洞察  
**目标用户：** ClawGlasses 硬件用户、AI 研究者、企业  
**核心价值：** 用户众包上传真实世界图片 → 数据清洗 → 视觉模型训练  
**变现模式：** 数据销售、API 调用、企业合作

---

## 2. 目标用户

| 用户类型 | 场景 | 需求 |
|----------|------|------|
| ClawGlasses 用户 | 日常佩戴拍摄 | 上传图片赚积分/代币 |
| AI 研究团队 | 获取训练数据 | 高质量、多样化的图片数据集 |
| 企业客户 | 商业洞察、行业分析 | 定制化数据 + 分析报告 |
| 开发者 | 构建 AI 应用 | API 调用、数据标注工具 |

---

## 3. 核心功能

### 3.1 众包上传系统

| 功能 | 描述 |
|------|------|
| ClawGlasses 直传 | 硬件直连，一键上传 |
| 手机 App 上传 | iOS/Android 客户端 |
| Web 上传 | 拖拽上传批量图片 |
| 自动标注 | AI 自动打标签（场景、物体、情绪） |
| 质量审核 | 多级审核确保数据质量 |

### 3.2 数据管理系统

| 功能 | 描述 |
|------|------|
| 数据存储 | IPFS + 区块链确权 |
| 数据清洗 | 去重、过滤、质量分级 |
| 标注工具 | 矩形框、语义分割、关键点 |
| 版本管理 | 数据集版本控制 |

### 3.3 视觉模型训练

| 功能 | 描述 |
|------|------|
| 数据集市场 | 公开数据集买卖 |
| 训练 pipeline | 自动化训练流程 |
| 模型评估 | Benchmark 对比 |
| API 服务 | 推理 API |

### 3.4 商业洞察（数据分析）

| 功能 | 描述 |
|------|------|
| 行业趋势 | 上传图片分析热门场景 |
| 消费者洞察 | 场景、行为分析 |
| 竞品监控 | 市场视觉趋势 |
| 报告生成 | 自动生成洞察报告 |
