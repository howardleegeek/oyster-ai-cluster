# OysterWorld PRD - OysterRepublic 空间版

> **Version:** 1.0  
> **Date:** 2026-02-14  
> **Project:** OysterWorld (claw-nation)  
> **Goal：** OysterRepublic.xyz 的空间/地图版本，众包世界图像数据

---

## 1. 产品概述

**产品名称：** OysterWorld  
**产品定位：** 去中心化世界摄像头网络 + 空间数据平台  
**目标用户：** OysterRepublic 社区成员、摄像头节点运营者、数据消费者  
**核心价值：** 用户部署摄像头节点 → 众包真实世界图像 → 数据变现  
**变现模式：** 数据销售、节点激励、空间分析服务

---

## 2. 目标用户

| 用户类型 | 场景 | 需求 |
|----------|------|------|
| 节点运营者 | 部署 ClawGlasses / 摄像头 | 获取 $WORLD 代币激励 |
| OysterRepublic 成员 | 已有 Passport NFT | 参与节点网络 |
| 数据买家 | 城市规划、零售分析 | 购买空间数据 |
| 开发者 | 构建基于位置的应用 | API 调用 |

---

## 3. 核心功能

### 3.1 节点网络

| 功能 | 描述 |
|------|------|
| 节点注册 | 摄像头/ClawGlasses 注册到网络 |
| 实时推流 | 摄像头实时上传图像 |
| 位置证明 | GPS + 区块链确权 |
| 节点激励 | $WORLD 代币分发 |

### 3.2 空间数据

| 功能 | 描述 |
|------|------|
| 世界地图 | 全球节点分布可视化 |
| 单元格系统 | H3 地理网格划分 |
| 事件追溯 | 任意位置的历史图像 |
| 热点分析 | 人流/热度图 |

### 3.3 数据服务

| 功能 | 描述 |
|------|------|
| 历史查询 | 任意坐标的历史图像 |
| 实时监控 | 实时视频流 |
| 区域报告 | 指定区域的统计数据 |
| API 开放 | 开发者接入 |

### 3.4 与 OysterRepublic 整合

| 功能 | 描述 |
|------|------|
| Passport 绑定 | 节点绑定到 Passport NFT |
| 激励分发 | $WORLD 代币空投 |
| 社区入口 | 从 oysterrepublic.xyz 跳转 |
