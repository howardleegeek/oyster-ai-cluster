## 目标
重构内容管理页面 (Content, ContentDetail, ContentNew)，打造内容编辑器的高级感

## 约束
- 不改后端 API 逻辑
- 不改功能流程，只提升视觉

## 具体改动

### Content.tsx (内容列表)
- **美学方向**: 内容库 (Content Library)
- **设计要点**:
  - 筛选器: 下拉菜单，优雅的 dropdown
  - 内容卡片: 缩略图 + 标题 + 元信息
  - 分页: 简洁的页码导航

### ContentDetail.tsx (内容详情)
- **美学方向**: 与 Content 统一
- **设计要点**:
  - 预览区域: 大幅内容预览
  - 侧边栏: 元数据编辑

### ContentNew.tsx (新建内容)
- **美学方向**: 创作空间 (Creative Space)
- **设计要点**:
  - 表单: 宽松的 spacing，输入框优雅
  - 富文本编辑器区域: 简洁的工具栏

## 验收标准
- [ ] 内容列表清晰美观
- [ ] 新建/编辑表单易用
- [ ] 整体风格统一
