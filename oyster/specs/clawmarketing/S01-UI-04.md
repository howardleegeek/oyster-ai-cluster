## 目标
重构营销活动页面 (Campaigns, CampaignDetail)，打造活动管理的高级感

## 约束
- 不改后端 API 逻辑
- 不改功能流程，只提升视觉

## 具体改动

### Campaigns.tsx
- **美学方向**: 活动控制台 (Campaign Console)
- **设计要点**:
  - 页面标题: 活动数量 badge
  - 活动列表: 表格或卡片，状态 tag (活跃/草稿/结束)
  - 状态 tag: 颜色区分，优雅的圆角 pill
  - 操作按钮: 编辑/删除，icon + text
  - 新建按钮: 醒目的 CTA

### CampaignDetail.tsx
- **美学方向**: 与 Campaigns 统一
- **设计要点**:
  - 活动概览: 头部信息 banner
  - 数据展示: 图表区域
