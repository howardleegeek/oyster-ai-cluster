## 目标
重构 ClawMarketing 核心页面 (Dashboard, Home)，打造数据仪表盘的高级感

## 约束
- 不改后端 API 逻辑
- 不改功能流程，只提升视觉
- 使用现有依赖

## 具体改动

### Dashboard.tsx
- **美学方向**: 数据仪表盘专业感 (Data Dashboard Pro)
- **设计要点**:
  - 侧边栏: 深色侧边栏，图标 + 文字，hover 优雅高亮
  - 顶部: 简洁 header，通知图标，用户头像
  - 统计卡片: 3-4 个 KPI 卡片，each 有独特的 accent color
  - 卡片: 轻微阴影，圆角，数据可视化区域
  - 动画: 数字滚动 count-up 效果

### Home.tsx
- **美学方向**: 与 Dashboard 统一
- **设计要点**:
  - Hero 区域: 大标题 + 副标题，渐变文字效果
  - 功能卡片: 网格布局，hover 上浮 + 阴影加深
  - 布局: 充足的 negative space

## 验收标准
- [ ] Dashboard 有数据仪表盘的专业感
- [ ] Home 与 Dashboard 风格统一
- [ ] KPI 卡片有视觉层次
- [ ] 动画流畅
