## 3. 核心功能

### 3.1 TELOS 品牌目标系统

每个品牌/项目有一个 TELOS 配置：

- **Mission** - 品牌使命（一句话描述）
- **Goals** - 营销目标（涨粉？转化？曝光？）
- **Voice** - 品牌调性（专业？幽默？温暖？）
- **Strategy** - 策略方向（内容日历、发布节奏）
- **Hooks** - 自动化钩子（新品发布、活动触发）

### 3.2 AI Agent 工作流

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  Analyst    │ →  │   Writer    │ →  │  Reviewer   │ →  │ Publisher  │
│  分析需求    │    │  生成内容   │    │  审核质量   │    │  发布到平台 │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
```

- **Analyst** - 分析需求，输出内容 brief
- **Writer** - 基于 brief 生成内容（支持多平台：Twitter、Discord、Blog）
- **Reviewer** - 审核内容质量、品牌调性一致性
- **Publisher** - 自动发布到目标平台

### 3.3 平台集成

| 平台 | 功能 | 状态 |
|------|------|------|
| Twitter/X | 发帖、回复、刷数据 | MCP 自动化 |
| Discord | 频道消息、 webhook | MCP 自动化 |
| Blog/官网 | CMS 集成 | 待开发 |
| 邮件营销 | SendGrid 集成 | 待开发 |

### 3.4 记忆系统

三层记忆，AI 学习历史表现：

- **热记忆** - 最近 7 天帖子效果（点赞/回复/转化）
- **温记忆** - 30 天规律总结（什么类型内容效果好）
- **冷记忆** - 原始数据存档

### 3.5 数据分析

- 发布效果看板
- ROI 追踪
- 竞品对比
- AI 表现评分
