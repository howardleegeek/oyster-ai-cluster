## 目标
重构分析页面 (Analytics, Scout, Scheduler)，打造数据可视化的高级感

## 约束
- 不改后端 API 逻辑
- 不改功能流程，只提升视觉

## 具体改动

### Analytics.tsx (分析)
- **美学方向**: 数据指挥中心 (Data Command Center)
- **设计要点**:
  - 图表区域: 多个图表卡片，错落排列
  - 图表: 使用 chart 库，优雅的颜色配置
  - 数据表格: 斑马纹，hover 高亮

### Scout.tsx (侦察)
- **美学方向**: 与 Analytics 统一
- **设计要点**:
  - 搜索区域: 大搜索框，icon 装饰
  - 结果卡片: 信息丰富的列表项

### Scheduler.tsx (调度)
- **美学方向**: 时间轴 (Timeline)
- **设计要点**:
  - 日历/时间表: 清晰的网格
  - 事件卡片: 颜色区分状态

## 验收标准
- [ ] 数据可视化美观
- [ ] 图表配色优雅
- [ ] 布局有层次感
