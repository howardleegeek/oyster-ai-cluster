## 目标
重构工具类页面 (Personas, PersonaEditor, AgentControl, TelosEditor, Accounts)，打造工具面板的高级感

## 约束
- 不改后端 API 逻辑
- 不改功能流程，只提升视觉

## 具体改动

### Personas.tsx (人物画像列表)
- **美学方向**: 人物画廊 (Persona Gallery)
- **设计要点**:
  - 人物卡片: 头像 + 名称 + 标签，网格布局

### PersonaEditor.tsx (人物编辑)
- **美学方向**: 创作工作台
- **设计要点**:
  - 表单布局: 左侧配置，右侧预览

### AgentControl.tsx (Agent 控制)
- **美学方向**: 控制台 (Console)
- **设计要点**:
  - Agent 列表: 状态指示器 (在线/离线)
  - 控制按钮: 启动/停止，明显的状态切换

### TelosEditor.tsx (Telos 编辑)
- **美学方向**: 与 PersonaEditor 统一

### Accounts.tsx (账户)
- **美学方向**: 账户设置 (Account Settings)
- **设计要点**:
  - 设置表单: 分组，section 分隔

## 验收标准
- [ ] 所有工具页面风格统一
- [ ] 交互状态清晰
