## 目标
重构品牌管理页面 (Brands, BrandDetail)，打造品牌列表的高级感

## 约束
- 不改后端 API 逻辑
- 不改功能流程，只提升视觉

## 具体改动

### Brands.tsx
- **美学方向**: 品牌画廊感 (Brand Gallery)
- **设计要点**:
  - 页面标题: 大字 + 描述，渐变装饰线
  - 品牌卡片: 网格布局，品牌 logo + 名称 + 描述
  - 卡片 hover: 上浮 + 边框发光
  - 空状态: 优雅的插画或图标

### BrandDetail.tsx
- **美学方向**: 与 Brands 统一
- **设计要点**:
  - 品牌头部: 大 banner，品牌信息
  - 内容分区: 清晰的 tab 或 section 区分

## 验收标准
- [ ] 品牌列表有画廊感
- [ ] 品牌详情页布局清晰
- [ ] 卡片交互有反馈
