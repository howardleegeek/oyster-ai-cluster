#### 目标
实现用户登录 API 并集成 JWT 认证。

#### 技术方案
1. 在 `main.py` 中添加一个 `/login` 路由，处理用户登录请求。
2. 在 `models/user.py` 中添加用户认证逻辑，使用 `passlib` 进行密码哈希。
3. 在 `main.py` 中集成 JWT 认证，使用 `python-jose` 生成和验证 JWT。
4. 在 `tests/test_main.py` 中编写测试用例，验证登录和认证功能。
