## 目标
实现用户注册 API 的基础功能。

## 技术方案
1. 在 `main.py` 中添加一个 POST 端点 `/api/users` 来接收用户注册数据。
2. 在 `models.py` 中定义 `User` 数据模型，包含必要字段如用户名、邮箱和密码。
3. 在 `tests/test_main.py` 中编写测试用例，确保 API 能够正确处理有效和无效的注册请求。
