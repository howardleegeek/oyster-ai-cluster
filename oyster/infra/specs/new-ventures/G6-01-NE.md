## 目标
集成一个基于AI的推荐系统来个性化用户内容展示。

## 技术方案
1. 使用 Hugging Face 的 Transformers 库加载预训练的推荐模型。
2. 在 FastAPI 的主路由中添加一个端点来处理用户请求并返回推荐结果。
3. 修改现有的用户模型以包含推荐相关的字段。
4. 编写一个辅助函数来处理模型推理和结果格式化。
5. 在 `main.py` 中添加相应的 API 路由。
