## 目标
实现基于 JWT 的身份验证和授权。

1. 使用 `PyJWT` 库生成和验证 JWT 令牌。
2. 在用户登录时生成 JWT 令牌，并设置适当的过期时间。
3. 在受保护的 API 端点中验证 JWT 令牌。
4. 在 `authentication.py` 中实现 JWT 生成和验证逻辑。
5. 编写基础测试以验证身份验证和授权流程。
