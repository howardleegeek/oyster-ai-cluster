## 目标  
实现多平台发布功能，支持将帖子发布到多个社交媒体平台。  

## 技术方案  
1. 在 `poster_publisher.py` 中实现一个发布器类，支持 Bluesky、Twitter 和 Facebook 等平台。  
2. 使用各平台的 API 库（如 `bluesky` 和 `tweepy`）进行集成。  
3. 实现一个方法，接受帖子内容和目标平台列表，依次发布到每个平台。  
4. 在 `tests/test_publisher.py` 中编写测试用例，验证帖子是否能正确发布到各个平台。
