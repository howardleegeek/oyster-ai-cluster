## 目标  
在用户界面中实现保险购买表单，允许用户输入必要的信息并提交购买请求。  

## 技术方案  
- 在 `InsuranceForm.jsx` 中创建保险购买表单，包括保费、赔付金额等输入字段。  
- 添加表单验证，确保用户输入的数据有效。  
- 在 `App.jsx` 中集成表单组件，并处理表单提交事件，调用后端 API 购买保险。  
- 使用 React 状态管理来显示提交结果和错误信息。
