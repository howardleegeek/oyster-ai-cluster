## 目标
集成自动化交易机器人，用户可以设置交易策略并自动执行。

## 技术方案
1. 在 `src/trading/bot.py` 中开发一个交易机器人，能够根据用户设定的策略进行自动交易。
2. 在 `src/api/trading.js` 中添加 API 端点，用于接收用户设定的交易策略和启动/停止机器人的指令。
3. 使用 Node.js 后端处理交易机器人的逻辑和与交易所的通信。
4. 实现用户界面，允许用户配置和监控交易机器人。
