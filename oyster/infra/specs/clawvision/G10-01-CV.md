## 目标
在 FastAPI 中实现基于 JWT 的身份验证机制。

## 技术方案
1. 使用 `PyJWT` 库处理 JWT 令牌。
2. 在 API 路由中添加依赖项以验证 JWT。
3. 创建一个新的中间件来验证每个请求的 JWT。
4. 在 `security.py` 中编写辅助函数来生成和验证 JWT。
5. 编写单元测试以验证 JWT 的生成和验证功能。

## 约束
- 在现有代码库内修改，不新建超过3个文件。
- 不留 TODO/FIXME/placeholder。
- 编写基础测试以覆盖 JWT 生成和验证。
