## 目标

## 技术方案
- 在 `tests/` 目录下添加单元测试、集成测试和端到端测试，确保代码的各个部分都经过充分测试。
- 配置 CI 工具（如 GitHub Actions 或 GitLab CI），在每次代码提交时自动运行测试。
