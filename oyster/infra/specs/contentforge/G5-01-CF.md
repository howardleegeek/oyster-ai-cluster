## 目标
实现基本的社交媒体内容生成功能，支持文本和图片的生成。

## 技术方案
1. 在 `app/main.py` 中添加一个新的 API 端点 `/generate-content`，接受请求参数如内容类型（文本或图片）和主题。
2. 在 `app/models.py` 中定义一个新的数据模型 `GeneratedContent` 来存储生成的内容。
3. 使用 FastAPI 的依赖注入功能来处理内容生成逻辑。
4. 在 `tests/test_main.py` 中编写基础测试，确保端点返回正确的数据格式和状态码。
