## 目标

## 技术方案
1. 在 `app/main.py` 中添加发表评论和获取评论的 API 端点。
2. 在 `app/models/comment.py` 中定义评论模型，包括评论内容、作者、帖子 ID 和发布时间字段。
3. 在 `app/main.py` 中实现获取帖子下所有评论的逻辑。
4. 在 `tests/test_comment.py` 中编写测试用例，验证发表评论和获取评论的 API 端点。
