## 目标
实现用户通知功能，通知用户关于评论和点赞的信息。

## 技术方案
1. 在 `app/main.py` 中添加获取用户通知的 API 端点。
2. 在 `app/models/notification.py` 中定义通知模型，包括通知类型、相关帖子 ID、发送者 ID 和发送时间字段。
3. 在用户对帖子进行评论或点赞时，自动生成相应的通知。
4. 在 `tests/test_notification.py` 中编写测试用例，验证通知的生成和获取。
