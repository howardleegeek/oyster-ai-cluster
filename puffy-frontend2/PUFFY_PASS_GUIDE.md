## ðŸ”§ Troubleshooting

### "Insufficient SOL Balance"

- Ensure you have at least 0.031 SOL plus additional SOL for transaction fees
- Recommend preparing 0.05 SOL to ensure transaction success

### "Wallet Connection Failed"

- Refresh the page and retry
- Ensure wallet app is updated to the latest version
- Mobile: Ensure you return to wallet browser from the correct app

### "Invalid Minting Code"

- Confirm you're using the correct exclusive link
- Verify you've completed the Matrica verification process in Discord
- Ensure the minting code hasn't been used before
- If link expires, get a new link from Discord

### "Invalid Invitation Code"

- Carefully check for spelling errors in the invitation code
- Contact the invitation code provider to confirm validity
- Check if network is effective

### "NFT Still Not Appearing After 3+ Minutes"

- Check using Solflare or Backpack wallet
- If transaction shows successful but NFT is missing, contact technical support

### "Duplicate Payment"

- If charged multiple times due to network issues or transaction errors, our team will check transaction history during business hours
- We will refund excess minting fees to your original wallet

### Mobile-Specific Issues

- **Page turns white**: Pull down to refresh page
- **Can't find back button**: Use system task switcher
- **Connection lost**: Reconnect wallet, status will be preserved
- **Page stuck**: Force refresh browser page
